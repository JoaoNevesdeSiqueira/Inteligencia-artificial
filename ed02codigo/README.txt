
Instruções para rodar os códigos dos labirintos no Visual Studio Code
(Mesmo processo em qualquer compilador)

1. Pré-requisitos
   -Tenha o Python instalado na sua máquina (preferencialmente versão 3.12 que foi a utilizada).
   -Instale o Visual Studio Code (VS Code) e a extensão do Python da Microsoft.
   -Certifique-se de que o Python está adicionado às variáveis de ambiente (PATH).

2. Organização da pasta
   -Crie uma pasta com todos os arquivos Python (labirinto1.py, labirinto2.py, até labirinto6.py).
   -Dentro da mesma pasta, coloque os arquivos .txt dos labirintos (maze1.txt, maze2.txt, até maze6.txt).

3. Abrir no VS Code
   -Abra o VS Code.
   -Vá em Arquivo > Abrir Pasta e selecione a pasta onde estão os arquivos.
   -Abra um dos arquivos .py no editor.

4. Executar o código
   -Com o arquivo aberto, clique com o botão direito e selecione Run Python File in Terminal.
   -O terminal do VS Code será aberto e o script será executado.
   -O script vai mostrar o tempo de execução, uso de memória e número de passos para resolver o labirinto.

5. Possíveis erros
   -Se der erro de "FileNotFoundError", verifique se o arquivo .txt está no mesmo diretório que o .py.
   -Se aparecer "python não é reconhecido", reinstale o Python e marque a opção "Add to PATH" durante a instalação.
